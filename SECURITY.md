# Security Policy

## Supported Versions

Since this is a research Proof-of-Concept, only the latest version on the `main` branch is supported.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in Jira Time Machine, please report it privately.

**DO NOT** open a public issue on GitHub for sensitive security vulnerabilities.

### How to Report

Please report vulnerabilities by emailing the repository owner directly or using the [GitHub Private Vulnerability Reporting](https://github.com/petripaananen/Jira-Time-Machine/security/advisories/new) feature if enabled.

### Response Timeline

We will endeavor to acknowledge your report within 48 hours and provide an estimated timeline for a fix.
